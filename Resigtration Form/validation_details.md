# Student Registration Form Validation Details

## Fields With Validation

1. **First Name** → Cannot be empty
2. **Last Name** → Cannot be empty
3. **Date of Birth** → Cannot be empty
4. **Gender** → One option must be selected
5. **Blood Group** → Must select an option
6. **Nationality** → Cannot be empty
7. **Category** → Must select an option
8. **Aadhar Number** → Must contain exactly 12 digits
9. **Email** → Must follow valid email format
10. **Phone Number** → Must contain exactly 10 digits
11. **Emergency Contact Name** → Cannot be empty
12. **Emergency Contact Phone** → Must contain exactly 10 digits
13. **Address Line 1** → Cannot be empty
14. **City** → Cannot be empty
15. **State** → Cannot be empty
16. **Pincode** → Must contain exactly 6 digits
17. **Course** → Cannot be empty
18. **Department** → Cannot be empty
19. **Year of Study** → Cannot be empty
20. **Semester** → Cannot be empty
21. **Roll Number** → Cannot be empty
22. **Registration Number** → Cannot be empty
23. **Previous School** → Cannot be empty
24. **Previous Board** → Cannot be empty
25. **Percentage** → Cannot be empty
26. **Father Name** → Cannot be empty
27. **Father Occupation** → Cannot be empty
28. **Father Phone** → Must contain exactly 10 digits
29. **Mother Name** → Cannot be empty
30. **Annual Income** → Cannot be empty
31. **Hostel Required** → One option must be selected
32. **Transport Required** → One option must be selected
33. **Scholarship** → One option must be selected
34. **Laptop Available** → One option must be selected
35. **Internet Access** → One option must be selected
36. **Declaration Checkbox** → Must be checked before submission
